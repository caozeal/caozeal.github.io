自定义 API 接入及安装指南

前提条件： 确保系统已安装 Node.js 18.x 或更高版本。准备好您的自定义 API 访问令牌。
1

安装 Node.js

推荐使用 winget 或直接从 nodejs.org 下载安装 LTS 版本。

winget install OpenJS.NodeJS.LTS
2

全局安装 Claude Code

npm install -g @anthropic-ai/claude-code
3

配置 API 代理接口

编辑 ~/.claude/settings.json (或相应平台的配置文件地址):

{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "您的_API_令牌",
    "ANTHROPIC_BASE_URL": "https://api.your-proxy.com"
  }
}
